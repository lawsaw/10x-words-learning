### Mandatory project requirements:

- User access control mechanism appropriate for the type of application (e.g., login screen)
- Data management — ability to create, read, update, and delete items (typical CRUD functionality)
- Business logic
- PRD and other contextual documents from modules 2 and 3
- Tests — at least one test verifying the application’s behavior from the user’s perspective
- CI/CD pipeline — process for building the application and running tests
- The project is available as an installable package with Docker
